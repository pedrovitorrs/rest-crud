## How to Install

* To download the project follow the instructions bellow:

```
 git clone https://github.com/pedrovitorrs/rest-crud.git
 cd rest-crud
```

* Install the dependencies and start the project:

```
 yarn install
 yarn start
```

or

```
 npm install
 npm start
